# dht
**join dht network,get infohash,query torrent metadata.**
## 快速使用
- php>=7.1.0
- 安装[Swoole](https://www.swoole.com/) 扩展
- 使用 composer

```
git clone https://github.com/devchen-org/dht.git
cd dht
composer install

```
- 运行

```
php artisan dht:spider

```
